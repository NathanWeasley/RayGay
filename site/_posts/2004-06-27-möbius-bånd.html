Nedenstående er bare lidt leg med hhv. 100 og 5000 cylindere.

Objektet er programmeret således:

$cyl = group {
    sphere { $chrome 20 <0,200,0> }
    cylinder { $chrome 20 <0,-200,0> <0,200,0> }
    sphere { $chrome 20 <0,-200,0> }
}

$i = 0
$num = 100
$windings = 3.5
do {
    object $cyl
       rotate { <1,0,0> $windings*$i*(360/$num) }
       translate <0,250,500> 
       rotate { <0,1,0> $i*(360/$num) }
    $i++
} while ($i < $num)
