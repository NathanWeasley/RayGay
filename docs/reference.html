Vector operations

vdot

(vdot vector vector) → real








vcross


(vcross vector vector) → vector









rotate


(rotate vector axis angle) → vector


(rotate object axis angle) → object









translate


(translate vector translation) → vector


(translate object translation) → object







Paths





make-circle


(make-circle center radius normal) → path


(make-circle '(0 0 0) 10 '(0 0 1))








make-ellipse


(make-ellipse center radius-x radius-y normal) → path


(make-ellipse '(0 0 0) 10 20 '(0 0 1))








make-spiral


(make-spiral path radius windings offset) → path









make-bezierspline


(make-bezierspline vectors) → path


(make-bezierspline ('(0 0 0) '(10 10 10) '(20 20 10)))








point-on-path


(point-on-path path t) → vector









tangen-to-path


(point-on-path path t) → vector
